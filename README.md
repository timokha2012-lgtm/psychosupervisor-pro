## ✅ Реализованные функции

### 🎭 Тренажёр диалога
- 18 психотипов клиентов (зависимости, тревога, депрессия, ПТСР, травма, духовный кризис)
- Динамическое поведение клиента: тревога, доверие, открытость меняются в реальном времени
- Фазы сессии: контакт → исследование → инсайт → изменение
- AI-клиент через OpenRouter (Claude, Gemini, GPT) — реалистичные ответы

### 🤖 AI-Супервизор (OpenRouter)
- Подсказки в реальном времени во время сессии
- Полный разбор сессии после завершения (ошибки + сильные стороны)
- Прямые вопросы супервизору через кнопку 🧠
- Детекция красных флагов (суицидальный риск и др.)
- Рекомендация техник по фазе и состоянию клиента

### 📝 Анализ транскрипций
- Загрузка текстовой транскрипции — полный разбор
- **Web Speech API** — запись голоса в реальном времени (Chrome)
- AI-анализ транскрипта через OpenRouter
- Скриптовый анализ (без API-ключа)

### 🔑 Система продажи доступа
- Ключи формата `PSP-XXXX-XXXX-XXXX`
- 3 тарифа: Студент (490₽/30 дн), Профессионал (1490₽/90 дн), Команда (4990₽/год)
- Генерация ключей через консоль браузера
- Демо-ключ для ознакомления: `PSP-DEMO-2026-FREE`

### 📊 Прогресс и геймификация
- 7 уровней: Стажёр → Слушатель → Практик → Консультант → Специалист → Мастер → Эксперт
- XP за каждую сессию, система бейджей (13 достижений)
- Графики динамики оценок и радар навыков (Chart.js)

### 📚 База знаний
- 24 карточки: техники (КПТ, МИ, DBT, ACT, Роджерс), ошибки новичков, духовный подход
- Поиск и фильтрация по категориям
